DOM Selectors
Introduction
DOM selectors are used to select HTML element within a document using javascript
There are 5 types of selectors
  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()
To use these selectors we prefix those element with document object
  • document.getElementsByTagName()
  • document.getElementsByClassName()
  • document.getElementById()
  • document.querySelector()
  • document.querySelectorAll()
getElementsByTagName
This returns all the element that matches the specified Tag name
For example

document.getElementsByTagName('h1') return all the elements that matches the tag name h1
getElementsByClassName
This method returns all the element that matches specified class name
For example
document.getElementsByClassName('item') return all the elements that matches the class name item.
getElementById
In this selection of element depend on Id name . This returns only the first matched element while ignoring rest.
For example
document.getElementsById('list-items') return specific element that matches the id name list-items.
querySelector
This return the first element that matches to specified CSS selector either class name or id names
To access element which define as id
document.querySelector('#id-name')
To access element which define as class-name
document.querySelector('.class-name')
querySelectorAll
This will return all the elements that matches to the specified CSS selector in the DOM
It returns all elements having that id or class-name.
These are some basic information of DOM selectors.
